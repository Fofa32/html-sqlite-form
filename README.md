# نموذج HTML + قاعدة بيانات SQLite

## خطوات التشغيل المحلي:
1. ثبت الحزم:
```
npm install
```

2. شغّل السيرفر:
```
npm start
```

3. افتح المتصفح:
```
http://localhost:3000
```

## طريقة الرفع أونلاين:
- أنشئ حساب على [https://render.com](https://render.com)
- اربط GitHub بالموقع، وارفع المشروع هناك.
- اختر "Web Service"
- في إعدادات التشغيل، اكتب:
```
npm install && npm start
```
